# Ecommerce-website
A full-featured e-commerce website for electronics where admins can manage products and users can browse, add to cart, and purchase items.


Features:
Admin Panel:
Register and login securely
Add, update, and delete products
Manage inventory

User Panel:
Register and login
Browse products by categories
Add products to cart
Checkout and place orders

Other Features
Secure session management
Responsive design for desktop and mobile
Real-time product updates

Tech Stack:
Frontend: HTML, CSS, JavaScript
Backend: PHP
Database: MySQL

Usage:
Clone the repository
Import the database from database.sql

Update config/db.php with your database credentials

Open the project in your local server (XAMPP/WAMP)
